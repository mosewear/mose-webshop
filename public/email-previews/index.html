Vijf complete herontwerpen van de OrderConfirmation email op basis van de MOSE brand identity: Anton (display) + Montserrat (body), brand-primary #00A676, pure zwart en wit, harde edges (geen border-radius). Alle designs gebruiken table-based layouts, inline CSS, MSO conditional comments voor Outlook, responsive media queries, en graceful font-fallbacks. Klaar voor Gmail, Apple Mail, Outlook, iOS Mail en alle andere clients.

Compatibiliteit-checklist (alle 5 designs)
  • ✓ Table-based layout (geen flex/grid CSS) — werkt in Outlook 2007-2021
  • ✓ Inline CSS op elk element + <style> voor media queries
  • ✓ MSO conditional comments voor VML-buttons in Outlook
  • ✓ Responsive mobile breakpoint < 620px (stacking + kleinere typografie)
  • ✓ Google Fonts (Anton + Montserrat) met Arial/Helvetica fallbacks voor Outlook
  • ✓ Preheader tekst voor inbox-preview
  • ✓ Max width 600px · veilig voor Gmail Web, Apple Mail, iOS, Android, Yahoo
  • ✓ width en height attributen op alle images
  • ✓ Alt-text op images · dark-mode meta tags
  • ✓ Geen border-radius (MOSE branding = harde edges, bonus: Outlook support)
